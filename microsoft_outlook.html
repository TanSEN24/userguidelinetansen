How to Setup Microsoft Outlook

Quick steps:

  1. Open Outlook and click File

    Open Outlook on your computer and click File in the top left corner.

  2. Click on Add Account

    Click Add Account to start setup.

  3. Type in your email address

    Type in the email address you want to add and click Connect.

  4. Enter your password

    Enter the password for your email account and click Connect to continue.

  5. Close automatic setup

    Click Done to complete the setup process.

  6. Go to File again

    Your email account is now set up, but you need to adjust a setting.

    Click File again in the top left corner.

  7. Click on Account Settings
    • Select the email from the drop-down bar.
    • Click Account Settings.
    • Select Account Name and Sync Settings.
  8. Set the Root folder path
    • Under Folder setting, type in INBOX.
    • Click Next to save your settings.
  9. You are done!

    Click Done and go to your inbox. You should now see folders properly aligned.
